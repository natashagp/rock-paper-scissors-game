# Rock Paper Scissors Game

### Step 1

Clone the repository

```shell
git clone https://github.com/natashagp/rock-paper-scissors-game.git
```

### Step 2

Change the directory to rock-paper-scissors-game

```shell
cd rock-paper-scissors-game
```

### Step 3

Run project

```shell
python main.py
```
